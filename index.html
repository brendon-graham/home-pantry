Home Pantry Setup

One-time Firebase configuration to enable real-time sync between phones

Step 1 — Create a Firebase project

  1. Go to console.firebase.google.com
  2. Click Add project → name it "Home Pantry" → Create
  3. Once created, click the Web icon (</>) to add a web app
  4. Register the app (name doesn't matter), skip Firebase Hosting
  5. Copy the firebaseConfig object shown

Step 2 — Enable Firestore

  1. In the left sidebar → Build → Firestore Database
  2. Click Create database → Start in test mode → Next → Enable
