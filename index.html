Home Pantry Setup

One-time setup — paste your Apps Script URL to connect to Google Sheets

Step 1 — Open the Google Sheet

Go to your Home Pantry Google Sheet → click Extensions → Apps Script

Step 2 — Paste the script

Delete any existing code, paste the contents of appsscript.js, then click Save (the floppy disk icon)

Step 3 — Deploy

  1. Click Deploy → New deployment
  2. Click the gear ⚙️ next to "Select type" → choose Web app
  3. Set Execute as: Me
  4. Set Who has access: Anyone
  5. Click Deploy → authorise when prompted
  6. Copy the Web app URL
